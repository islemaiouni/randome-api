# Meme API


Node API to return random programming meme images scrapped from [Memedroid](https://www.memedroid.com/memes/tag/programming).

## Setup

Install the dependencies


yarn i 

or npm i

npm start 

yarn start
```

## Tech Stack

Node, Puppeteer, Fastify

## API Reference

#### Get a random meme

```http
GET /
```
